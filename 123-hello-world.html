Step 1: Install RimStone
First, install RimStone.
Step 2: Build it
Create Hello World source file (hello.rim) in a new directory; note it's all one bash command:
echo 'begin-handler /hello public
    @Hello World!
end-handler' > hello.rim
Copied!

Create Hello World application:
rim -k helloworld
Copied!

Make Hello World application:
rim -q
Copied!

Step 3: Run it
You can run Hello World both as a service and from command line:
  • As a service, first start your Hello World application server:
    mrim -w 3 helloworld
    Copied!

    then connect to the service:
    rim -r --req="/hello" --silent-header --service --exec
    Copied!

  • Execute as command-line program:
    rim -r --req="/hello" --silent-header --exec
    Copied!

Expected result
Hello World!
Copied!
